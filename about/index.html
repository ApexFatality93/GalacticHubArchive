Purpose

Why this site exists

The Galactic Hub wiki contains a large amount of documented player discoveries, but raw cargo tables are not ideal for browsing. This archive displays and filters that data with direct links back to the original wiki page, so the archive acts as a discovery interface rather than a reskin of the GH wiki.

Data Source

Where the archive data comes from

Every archive page is generated from data exported from the NMS Galactic Hub wiki on Miraheze. The exporter script queries the wiki API and writes JSON files that this website reads directly.

Notes

What to expect

This site reads from exported JSON files rather than live cargo tables in the browser. That means data updates when the export script is run, not instantly when a wiki page changes.
